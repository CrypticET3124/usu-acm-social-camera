# Challenges
How many stars can you collect?

## Tier 1 - Touch Grass ⭐
- Personalize title/handle/caption and other text
- Edit colors of one theme
- Create one custom sticker (emoji, text, or simple component)
## Tier 2 - Designer ⭐⭐
- Create a new overlay preset
- Create two custom stickers
- Make one of the custom stickers *interactive* (see like-button example) Ideas: Sunglasses 😎 on/off, Tap to change color
## Tier 3 - Power User ⭐⭐⭐ 
- Sticker that shows live time
- Sticker that shows device info (battery %, platform, screen size, etc.)
- Edit the overlay and give it a new layout (branding in top-right, caption at bottom, etc.)
## Tier 4 - Code Wizard ⭐⭐⭐⭐
- Add multiple overlays/layouts and add UI to the camerlay camera-pane/preview-pane controls for swapping between them
- Add an in-app gallery (in-memory) showing your last 5 captures
- Customize the sticker picker
## Tier 5 - "EL FUA" ⭐⭐⭐⭐⭐
- Instead of Expo GO, use Expo dev client to allow saving photos to device + real gallery using MediaLibrary

## Social Bonus ⭐⭐
- Using Expo GO, try someone else's app and *steal one idea*
- Show your app to two people, get a "feature request", and implement it
